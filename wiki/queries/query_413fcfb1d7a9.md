# 如果乙方工程進度落後超過百分之五，甲方可以採取哪些行動？

- Chat session: `chat_6116250574f1`
- Answer method: `openai_compatible_chat`
- Retrieval mode: `hybrid_qdrant`

## Answer
如果乙方工程進度落後超過百分之五，甲方可以採取以下行動：

1. **終止或解除契約**
   根據第十九條第三項，工程進度落後20%以上且日數達十日以上時，甲方可以書面通知乙方終止或解除契約部分或全部，且不補償乙方因此所生之損失。

2. **暫停付款**
   根據第十八條第二項，工程進度延誤超過百分之五時，甲方得暫停付款。這與目前的問題條件完全吻合。

3. **損害賠償請求**
   根據第十九條第三項及第六條，若因工程延誤導致甲方或客戶受損，甲方有權請求損害賠償。

4. **另覓廠商承攬**
   根據第十九條第三項及第八條第四項，契約終止或解除時，甲方有權將未完成之工作收回自辦或另外招商承攬。

5. **從應付給乙方的款項中扣抵**
   根據第十八條及第十二條，甲方得自乙方得請領之工程款、保證金或其他款項中扣抵因暫停付款所涉金額，或扣抵違約金與損害賠償。

**主要限制條件：**
- 終止契約的門檻較嚴格，需同時符合落後20%以上且達十日以上（第19條）
- 暫停付款只需進度延誤超過5%即成立（第18條）
- 損害賠償需證明實際損失與延誤之間的因果關係

**結論：**
在乙方工程進度落後超過5%的情況下，甲方最直接的行動是依據第十八條暫停給付該期款項，同時可根據第十九條之具體條件評估是否達成了終止契約的門檻。
