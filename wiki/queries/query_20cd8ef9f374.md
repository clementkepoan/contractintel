# 保險理賠金在什麼情況下會直接付給乙方而不是甲方？

- Chat session: `chat_bef34ebdf52f`
- Answer method: `langchain_ollama`
- Retrieval mode: `hybrid_qdrant`

## Answer
根據提供的合約條款，保險理賠金通常不會直接支付給乙方，而是由甲方處理。具體來說：

1. **保單條款**：
   - 第十五條第（二）點指出，乙方應將甲方、乙方、轉包商、分包商共同列為被保險人並加註受益人特約條款指定甲方為受益人。這意味著任何因工程事故引起的理賠金都應該支付給甲方。

2. **合約條款**：
   - 第十五條第（三）點進一步強調，乙方應依前開規定辦理保險，皆應將甲方、乙方、轉包商、分包商共同列為被保險人並加註受益人特約條款指定甲方為受益人。這確保了理賠金由甲方處理。

因此，在任何情況下，保險理賠金不會直接支付給乙方，而是會根據保單條款和合約規定支付給甲方。

**對應條款：**
- [C1] 第十五條 保險第（二）點及第（三）點

综上所述，保险理赔金不会直接支付给乙方，而是由甲方处理。
