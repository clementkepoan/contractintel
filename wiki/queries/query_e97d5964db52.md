# 不可抗力結束後如果乙方不繼續履約，甲方要等多久才能終止合約？

- Chat session: `chat_a40d449757aa`
- Answer method: `langchain_ollama`
- Retrieval mode: `hybrid_qdrant`

## Answer
根據合約中的相關條款，若不可抗力事件結束後，乙方仍無法履行契約義務，甲方可以在該不可抗力事件結束後的30日內終止合約。具體來說：

### 條款依據
- [C1] 第二十條 不可抗力（第一段）：
  - 任一方如因天災、人禍、事變或政府法令變更等不可抗力之情事發生，致無法如期履行本契約所約定之義務時，得免負遲延給付責任。但前述情事結束後，受影響之一方應繼續履行本契約。
  - 惟如前述因不可抗力而無法履行或遲延履行之情形已持續達30日以上者，甲方得以書面通知乙方終止本契約。

### 結論
根據上述條款，若不可抗力事件結束後，乙方仍無法履行契約義務，甲方可以在該不可抗力事件結束後的30日內終止合約。因此，甲方不需要等待更長時間即可行使終止權。

**對應條款**
- [C1] 第二十條 不可抗力（第一段）：惟如前述因不可抗力而無法履行或遲延履行之情形已持續達30日以上者，甲方得以書面通知乙方終止本契約。
